# Interactive Calendar Component

A polished, interactive calendar component designed with a focus on **clean UI, smooth UX, and scalable frontend architecture**.

---

## 🚀 Live Demo

* Live Link: https://tuf-calendar-gray.vercel.app/
* Video Demo Link: https://www.loom.com/share/d26e5bf225434917af0744a2ec0c8f20

---

## ✨ Features

* Day range selection with clear visual states
* Notes support for selected date range
* Persistent data using localStorage
* Smooth, purpose-driven animations
* Fully responsive (mobile + desktop)
* Optimized hero image transitions

---

## 🛠 Tech Stack

* React / Next.js
* TypeScript
* Tailwind CSS
* Framer Motion
* LocalStorage

---

## 🧠 Key Highlights

* Fixed 42-cell calendar grid for layout consistency
* Clean component-based architecture
* Range-based note persistence across multiple dates
* Optimized rendering using memoization
* Image preloading for smooth UX
